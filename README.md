# Интеллектуальный помощник выбора кухни

## Модель представления знаний
Продукционная модель (ЕСЛИ–ТО) + элементы онтологии.

## Реализовано:
- 4 типа кухонь
- 3 продукционных правила
- механизм исключения
- интерактивный интерфейс

## Деплой
1. Создать репозиторий на GitHub
2. Загрузить файлы
3. Settings → Pages → Deploy from branch → main
